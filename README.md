# 🚀 I Can To – Sign Language Translator App

> AI-powered mobile app that translates sign language from images into Arabic.

---

## 📌 Overview

**I Can To** is a smart mobile application designed to help bridge communication between sign language users and Arabic speakers.

The app uses an AI model to analyze images of sign language gestures and translate them into clear Arabic text.

⚠️ **Important Note:**
- This project is a **graduation project prototype**
- It is **NOT the final product**
- The app is still **under development**

---

## 🎯 Features

### 🔐 Authentication (Firebase)
- Login with Email & Password
- Register with Email & Password
- Google Sign-In
- Forgot Password

### 🚀 Guest Mode
- Users can skip login and use translation directly

### 🤖 AI Translation
- Translate sign language from images into Arabic text

### ⭐ Favorites
- Save translations to favorites
- Requires login to store data permanently

---

## 🛠️ Tech Stack

- Flutter
- Dart
- Firebase Authentication
- State Management: Cubit (Bloc)
- Architecture: MVVM + Repository Pattern
- AI Model Integration
